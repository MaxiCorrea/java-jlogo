# java-jlogo

### Descripción
Simple intérprete del lenguaje Logo.

### Caracteristicas
1- Permite ejecutar los siguientes comandos: 
  * girarder/girarizq.
  * avanzar/retroceder.
  * color/pintar/borrar.
  * ubicar/ocultar/mostrar.
  * bajarpincel/subirpincel.
  * circulo.
  * repetir.
  * insertar.
  * nuevo /abrir/guardar.
   
### Tecnologías
   * JavaSE7
   * Swing
   * XML
